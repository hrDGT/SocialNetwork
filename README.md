# Social Network

A basic template for a social network. It uses https://dummyjson.com/ as a source of data.

# Required to run
* Node.js [https://nodejs.org/en/download/]

# Main dependencies
* React ^19.2.4
* mobx-react ^9.2.1
* tanstack/react-router ^1.167.0
* tanstack/react-query ^5.90.21
* Material UI ^7.3.9

# Installation 
* Clone using the web URL with `git clone https://github.com/hrDGT/SocialNetwork.git`
* Navigate into the project folder with `cd ECommerce App`
* Install dependencies with `npm install`

# How to run
* Run `npm run dev` to run the project in development mode
 
# For developers
* Run `npm run build` to create dist/ folder with production build
* Run `npm run lint` to check for lint issues
* Run `npm run lint:fix` to fix lint issues
* Run `npm run type-check` to check fot TS issues
* Use `npm run test` to run Vitest specs

# Deploy
* Project is deployed. You can access it via https://hrd-socialnetwork.netlify.app/
